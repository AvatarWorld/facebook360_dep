def scripts.render.rename_and_resize_frames.main (   argv) 
Validates CLI flags and sets up a Watchdog to monitor an input directory to
rename and resize files as they arrive. Inputs are assumed to have the naming
scheme: path/to/file/prefix.000000.tif.
